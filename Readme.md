# 🍔 Burger Landing Page
Proyek ini adalah sebuah landing page responsif bertema restoran burger yang dibuat untuk melatih kemampuan HTML dan CSS. Proyek ini menekankan pada penggunaan Flexbox untuk mengatur tata letak yang bersih dan modern.

## 🚀 Fitur Utama
**Section Hero**: Menampilkan penawaran utama dengan gambar burger yang menggugah selera.

**Daftar Menu Varian**: Menggunakan sistem kartu (cards) untuk menampilkan 4 jenis burger unik:

* **Double Beef & Cheese Burger**

* **Black Burger**

* **Red Burger**

* **Brown Burger**

**Testimoni**: Kutipan inspiratif tentang burger dari Thomas Keller.

**Call to Action (CTA)**: Section ajakan untuk bergabung yang menonjol dengan latar belakang biru.

## 🛠️ Teknologi yang Digunakan
* **HTML5**: Struktur semantik untuk konten web.

* **CSS3**: Penggunaan Flexbox, Google Fonts, dan desain responsif.
